Les executable produites par la commande make se trouvent dans le repertoire exec:
projet_0 projet_1 testTable projet
Pour executer:
Partie 1:
./exec/projet_0
Partie 2:
./exec/projer_1 N où N est le nombre de processus à creer
Partie 3:
./exec/testTable 
Partie 4:
./exec/projet N où N est le nombre de nodes souhaité 

Les fichier objet generé par la commande make se trouvent dans le repertoire obj
